Audio directory for Mindful Farm.

Place the following audio files here:
- bgm_day.mp3       (background music, daytime loop)
- bgm_night.mp3     (background music, nighttime loop)
- sfx_click.mp3     (UI click sound)
- sfx_chime.mp3     (plant growth chime)
- sfx_footstep.mp3  (player footstep)
- sfx_splash.mp3    (water/pond splash)
- ambient_birds.mp3 (daytime bird ambiance)
- ambient_crickets.mp3 (nighttime cricket ambiance)

The game will function without audio files — sounds are optional.
